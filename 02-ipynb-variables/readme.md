# 파이썬 가상환경

## 가상환경

### 1. 가상환경 생성
```bash
python -m venv venv
```

### 2. 가상환경 활성화

```bash
#  Windows
.\venv\Scripts\Activate

# macOS/Linux
source venv/bin/activate

# 가상환경 비활성화
deactivate
```

### 3. 주피터 노트북 관련 모듈 설치

```
pip install jupyter

pip install ipykernel

pip install ipython
```
