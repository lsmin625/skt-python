# 파이썬 코딩 맛보기 (ChatGPT 활용)

## 에코 프로그램 (2)

### 1. 사용자가 입력한 내용을 그대로 출력하는 프로그램 (`echo.py`)

```python
user_input = input("입력: ")
print("출력:", user_input)
```

### 2. 에코 프로그램 개선하기

사용자가 문장을 반복적으로 입력할 수 있도록 하고, 끝내고 싶을 때는 !quit을 입력하면 프로그램이 종료되도록 개선

## 비밀번호 검증 프로그램 (`password.py`)

### 정규 표현식을 이용한 비밀번호 검증

파이썬 정규 표현식을 사용하여 비밀번호를 검증하는 코드를 작성

- 사용자가 입력하는 비밀번호를 정규 표현식을 사용하여 검증하고 규칙에 적합한지 출력
- 비밀번호의 조건은 최소 하나의 영문 소문자, 영문 대문자, 숫자 및 기호가 포함되어야 함
- 비밀번호 검증은 별도의 함수를 통해 구현
