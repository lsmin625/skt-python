# 파이썬 코딩 맛보기 (ChatGPT 활용)

## 테트리스

### 1. 테트리스 게임 만들기 (`tetris.py`)

파이썬 pygame 모듈을 이용해서 테트리스 게임 만들기

- 떨어지는 블록을 키보드 화살표 키를 이용해서 좌우로 이동
- 위 화살표는 블록의 방향을 시계방향으로 돌릴 수 있고, 스페이스 키를 누르면 바로 떨어지기, 아래 화살표는 빠르게 하단으로 움직이기
- 떨어진 블록은 하단에 차례로 쌓고 블록의 열이 맞춰지면 블록이 삭제되는 형식으로 기능을 처리

### 2. 필요한 모듈 가상환경에 설치하기

```bash
pip install pygame
```

### 2. 가상환경에서 코드 실행하기

```bash
python tetris.py
```

### 3. 코드 개선하기

- 화면 왼쪽 상단에 다음에 떨어질 블록 미리보기
- 화면 오른쪽 상단에 획득한 점수 보이기

### 4. 추가 기능 구현하기

- **레벨 시스템**: 점수가 일정 수준에 도달하면 레벨이 올라가고 블록 낙하 속도 증가
- **일시정지 기능**: `P` 키를 눌러 게임을 일시정지하고 다시 재개
- **게임 오버 화면**: 블록이 최상단에 도달하면 게임 오버 메시지와 최종 점수 표시
- **배경음악 및 효과음**: 블록 회전, 라인 삭제, 게임 오버 시 효과음 추가
- **홀드 기능**: `C` 키를 눌러 현재 블록을 저장하고 나중에 사용
- **그림자 표시**: 현재 블록이 떨어질 위치를 미리 표시
